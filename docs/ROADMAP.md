# Roadmap

## Fase 1: base Laravel y autenticación

- Base Laravel limpia y desplegable
- Configuración de entorno y despliegue
- Autenticación de usuarios
- Preparación de estructura para multicliente

## Fase 2: clientes, almacenes, ubicaciones y artículos

- Gestión de clientes
- Gestión de almacenes
- Gestión de ubicaciones
- Catálogo de artículos

## Fase 3: stock por lote, entradas, salidas y movimientos

- Stock por lote
- Entradas de mercancía
- Salidas de mercancía
- Trazabilidad de movimientos

## Fase 4: importación CSV/Excel Friesland

- Importación de datos maestros
- Validaciones de formato
- Trazabilidad de importaciones
- Procesamiento orientado a Friesland

## Fase 5: etiquetas, informes, auditoría y backups

- Generación de etiquetas
- Informes operativos
- Auditoría de acciones
- Estrategia de backups y recuperación
